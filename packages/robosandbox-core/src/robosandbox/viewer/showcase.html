RoboSandbox x Newton

Side-by-side showcase for a Newton-on-DGX Spark spike. The left window keeps the current RoboSandbox task UI intact. The right window is reserved for Newton's live web viewer, forwarded from dgxspark to your laptop and embedded into this layout.

Goal
Visible backend spike
Host
DGX Spark
Viewer
Newton Viser

Launch Flow

1. Start the local RoboSandbox viewer on this machine.
2. Run bash scripts/newton_dgxspark_showcase.sh to start Newton remotely and forward the viewer to a local port.
3. Open this page at /showcase and keep both panes visible during the demo.
