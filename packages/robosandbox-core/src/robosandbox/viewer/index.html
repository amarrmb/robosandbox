How to use

  1. Pick a task below — pick_ycb_mug is a good first try.
  2. Click Run to watch the agent plan & execute.
  3. Click Record before Run to save an MP4 + events.jsonl to ./runs/.
  4. Check Teleop to drive the arm yourself (WASD + gamepad).
  5. After a run, scrub the Inspector timeline to replay any frame.
